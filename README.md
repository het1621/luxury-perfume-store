Aura | Immersive Luxury Fragrance Store
Aura is a high-end, cinematic e-commerce experience built with Next.js 15, GSAP, and Tailwind CSS. It moves away from traditional static layouts to create a "digital boutique" feel through motion design, glassmorphism, and fluid transitions.

✨ Key Features
Cinematic Hero Experience: A scroll-tied parallax effect that blurs the primary perfume bottle image as the user explores, ensuring maximum focus on brand typography.

Horizontal Gallery: A custom GSAP-powered horizontal scroll section that showcases products with a "Grayscale-to-Color" hover interaction.

Persistent Aesthetic: A fixed-background architecture that maintains visual continuity from the landing page down to the footer.

Fluid UI Components:

Slide-out Cart: A frosted-glass shopping drawer integrated into the global navigation.

Glassmorphic Login: A centered, immersive authentication portal with heavy backdrop filters.

Admin Dashboard: A dark-mode analytics suite featuring custom CSS data visualizations (no heavy external libraries).

Performance First: Optimized with Next.js Turbopack for lightning-fast refreshes and smooth 60fps animations.

🛠️ Tech Stack
Framework: Next.js 15 (App Router)

Styling: Tailwind CSS (Utility-first, responsive design)

Animations: GSAP (ScrollTrigger & Context API for React)

Icons: Lucide React

Deployment: Vercel
